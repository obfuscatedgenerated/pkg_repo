# OllieOS Package Repo

Packages for OllieOS, installable with `pkg add <name>`

## Creating packages

To get started, use the [template package](https://github.com/obfuscatedgenerated/ollieos_test_pkg) to create a new repo for your package.

Consult the [API docs](https://ollieg.codes/docs/) for help creating your programs, and the [template package README](https://github.com/obfuscatedgenerated/ollieos_test_pkg#readme) to build the final package.

Then you can [submit a pull request](https://github.com/obfuscatedgenerated/pkg_repo/compare) to contribute your built package files to the repo!
